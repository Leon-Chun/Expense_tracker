## 介紹

具有會員系統，紀錄屬於自己的代辦事項，可以瀏覽，查看，新增，刪除詳細資訊。

### 功能

- 能夠進行註冊，登入，並紀錄生活事項。
- 查看所有事項
- 依分類瀏覽詳細資訊


## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

   ```bash
   npm install
   ```

4. 安裝完畢後，繼續輸入：

   ```bash
   npm run start
   ```

5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

   ```bash
   Listening on http://localhost:3000
   ```

6. 若欲暫停使用

   ```bash
   ctrl + c
   ```
7. 能夠使用預設好的種子資料。
   種子帳號是：user1@example.com
   預設密碼：12345678

   ```bash
   npm run seed
   ```
